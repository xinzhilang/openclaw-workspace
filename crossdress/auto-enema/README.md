## 电路接线图

### 电源分配
```
24V电源 ─┬─→ 24V→12V降压模块 ──→ 继电器COM（供气泵+电磁阀）
         │
         └─→ MOSFET(D+) ──→ 蠕动泵+(红线)
              MOSFET(S-) ──→ 蠕动泵-(黑线)
              泵另一极 ──→ 电源GND（回路）
```

### 树莓派引脚分配

```
树莓派 3B+ GPIO
══════════════════════════════════

GPIO 17 (PWM0)  → MOSFET SIG  → 24V蠕动泵调速
GPIO 27          → 继电器 IN1   → 12V充气泵
GPIO 22          → 继电器 IN2   → 12V电磁阀（水路开关）
GPIO 23          → 继电器 IN3   → 备用/加热（暂空）
GPIO 24          → 继电器 IN4   → 备用

GPIO 5           → 脚踏开关 IN（接GND触发）
GPIO 6           → 急停按钮 IN（接GND触发）
GPIO 25          → DS18B20 DATA（+ 4.7kΩ上拉到3.3V）

I2C1 (SDA=GPIO2, SCL=GPIO3) → LCD1602 I2C 模块

GPIO 12          → 编码器 CLK（旋转脉冲A）
GPIO 13          → 编码器 DT（旋转脉冲B）
GPIO 19          → 编码器 SW（按键按下）

GPIO 26          → 蜂鸣器（有源，高电平响）

GPIO 20          → 按键1（启动/确认）
GPIO 21          → 按键2（停止/返回）

5V              → LCD1602 VCC
3.3V            → 编码器 VCC + DS18B20 VCC
GND             → 所有GND共地
```

## 管路连接

```
水箱（温水） → 硅胶管 → 蠕动泵(入)
                         蠕动泵(出) → 电磁阀 → 硅胶管 → 充气肛塞中心接头
                                                              │
                                                      充气肛塞气囊 ← 12V充气泵
```

## 3D打印外壳设计

外壳设计为一个三分隔的盒子：

```
┌────────────────────────────────────┐
│  [ 水箱 2L ]    [ 控制面板 ]       │
│                  LCD 编码器 按键    │
│                                    │
│  [ 电子仓 ]                         │
│   树莓派 | 继电器 | MOSFET | 降压   │
│   蠕动泵 | 电磁阀 | 气泵           │
└────────────────────────────────────┘
```

### 打印建议
- 耗材：PETG（耐温、机械强度好）
- 层高：0.2mm
- 填充：20%
- 水箱区需防水涂层或内衬食品级PP袋
