## 四、采购后调试步骤

### 第一步：硬件搭建
```
树莓派
├→ GPIO PWM → MOSFET → 24V蠕动泵
├→ 继电器 → 12V充气泵
├→ 继电器 → 12V电磁阀
├→ I2C → LCD1602
├→ I2C → PCF8591 + 模拟压力传感器 × 2
├→ GPIO → YF-S401 流量计（中断输入）
├→ GPIO(OneWire) → DS18B20
├→ GPIO → 编码器
├→ GPIO → 按键 × 2
├→ GPIO → 蜂鸣器
└→ GPIO → 急停按钮
```

### 第二步：管路连接
```
水箱 → 蠕动泵 → YF-S401流量计 → 电磁阀 → T型三通#1 → 肛塞注水管
                                               │
                                         水路压力传感器

充气泵 → T型三通#2 → 肛塞充气管
               │
         气路压力传感器
```

### 第三步：软件流程
```
[插入肛塞]
    ↓
[按确认] → 气泵自动充气 → 气压达25kPa停
    ↓
[自动注水] → 实时显示水量/流速/水压 → 到量停
    ↓
[保持倒计时] → 气压监控（漏气补压/骤降报警）
    ↓
[蜂鸣完成] → 自动放气 → LCD显示数据
```

## 五、电气参数速查

| 设备 | 电压 | 电流 | 控制方式 |
|------|:----:|:----:|----------|
| 蠕动泵 | 24V | ~0.5A | PMW(GPIO18) → MOSFET |
| 充气泵 | 12V | ~0.3A | GPIO27 → 继电器 |
| 电磁阀 | 12V | ~0.2A | GPIO22 → 继电器 |
| 树莓派4B | 5V | ~0.8A | 独立供电 |

**电源分配：**
```
24V电源 ─┬→ MOSFET ─→ 24V蠕动泵
         └→ LM2596降压 ─→ 12V ─→ 继电器 → 充气泵
                                └→ 继电器 → 电磁阀
树莓派 ← 自带的5V/3A电源
```
