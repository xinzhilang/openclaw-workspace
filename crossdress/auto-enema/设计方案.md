#### 补压策略细化

补压不是简单的一口气充到目标值，而是智能脉冲：

```
检测到压力＜23kPa
  → 充气泵启动 0.5秒
  → 等待 0.5秒（压力稳定）
  → 读取气压值
    ├ 已＞25kPa → 停泵，OK
    ├ 仍＜23kPa → 再充 0.5秒（循环）
    └ 60秒内超过3次循环 → 漏气量过大 → 严重漏气报警
```

补压期间 **不打断正在进行中的注水或保持流程**，仅在后台自动处理，LCD上显示补压标记提示用户。

## 十、校准步骤

### 10.1 YF-S401 流量计校准

YF-S401 规格通常是每升 450 个脉冲（450 pulses/L = 0.45 pulses/ml）。

实际验证：
```
1. 进入"手动控制 → 注水测试"
2. 出水口放入量杯
3. 运行直到注入100ml（按流量计读数）
4. 用量杯测实际出水量
5. 计算修正系数 = 实际值 / 显示值
6. 保存修正系数到配置文件
```

### 10.2 蠕动泵流速校准

```
1. 进入手动控制 → 注水测试
2. 分别设泵速50%/70%/90%
3. 各跑60秒，记录出水量
4. 得到流速表：
   50% → XX ml/min
   70% → YY ml/min  
   90% → ZZ ml/min
5. 用于系统预估剩余时间
```

### 10.3 XGZP6847 零点校准

```
开机后自动采集大气压作为零点。
水路传感器在未通水时读取值为零点基准。
```
