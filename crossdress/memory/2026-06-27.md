# 2026-06-27 硬件调试日志

## ✅ 蜂鸣器问题解决
- **根因**: 5V 有源蜂鸣器模块 CMOS 阈值 ~3.5V，ESP32 GPIO 最高 3.3V 关不掉
- **方案**: 改用 IN/OUT 切换控制（OUT+LOW=响 → IN高阻=静音）
- 改 `core.py`: `buz()` 重写
- 改 `boot.py`: 去掉 GPIO18 初始化，自检也用 IN/OUT 切换
- `boot.py` 旧代码把 GPIO18 设 OUT+LOW 保持 → 上电一直响，改完正常
