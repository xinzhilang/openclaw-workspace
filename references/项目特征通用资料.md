## 一、拆除工程

### 1.1 门窗拆除
```
1.拆除门口原有老化故障的电动卷帘门和防盗门
```

### 1.2 地面拆除
```
1.200mm厚混凝土拆除
2.200mm厚灰土拆除
```

### 1.3 雨篷拆除
```
1.钢化玻璃雨篷及钢梁拆除
```

### 1.4 管道拆除
```
1.影响施工的废旧管道拆除
2.拆除后运至甲方指定地点
```

### 1.5 余方弃置
```
1.废弃料品种:建筑垃圾外运及消纳
2.运距:自行考虑
```

### 1.6 石膏板吊顶拆除
```
1.拆除原有石膏板面层（含腻子、乳胶漆层）
2.拆除覆面龙骨、横撑龙骨（50*20mm轻钢龙骨）
3.拆除承载龙骨、吊杆及连接件
4.拆除灯具、风口等附着设备（如有）
5.人工拆除，不得破坏主体结构层
6.拆除后清理基层，修补孔洞
7.建筑垃圾分类装袋，场内运输至指定堆放点
8.余方弃置：垃圾外运及消纳，运距自行考虑
```

### 1.7 墙面铲除腻子及涂料层
```
1.铲除墙面原有腻子层至水泥砂浆基层
2.铲除原有涂料（乳胶漆/墙纸/墙布）面层
3.铲除局部空鼓、起皮、粉化区域至坚实基层
4.人工铲除为主，墙角、线盒周边辅以电动铲刀
5.铲除后墙面清扫浮灰，清水冲洗
6.检查基层：空鼓处凿除后用水泥砂浆修补找平
7.建筑垃圾装袋，场内转运至指定堆放点
8.余方弃置：垃圾外运及消纳，运距自行考虑
```

## 四、地面工程

### 4.1 广场砖地面（室外）
```
1.30厚广场砖
2.30厚1：3干硬性水泥砂浆
3.140厚C30混凝土浇筑层
4.200厚破碎石垫层
5.素土夯实，密度≥93%
```

### 4.2 环氧地坪漆
```
1.基层处理,打磨吸尘处理
2.一道环氧高渗透底漆
3.无溶剂中涂砂浆层,起厚度,一次成膜后,找补大坑
4.涂刷环氧腻子两道,待干后,用无尘打磨机打磨吸尘处理
```

### 4.3 塑胶地板
```
1.地面清洁,确保干净平整
2.根据图纸和设计方案,确定地板布置方式和方向,使用绳线进行布线
3.涂刮胶水,用齿形刮板均匀地将胶水涂抹在地面与塑胶地板背部
4.铺贴塑胶地板
```

### 4.4 PVC卷材地板
```
1.品种：同质透心PVC卷材/多层复合PVC卷材
2.厚度：2.0-3.0mm
3.耐磨层：≥0.35mm（商用），≥0.55mm（医疗）
4.幅宽：2m
5.施工工序：基层自流平找平→试铺裁切→满刮专用胶→铺贴→滚压排气→焊线
6.拼缝：热熔焊接（同色焊线）
7.踢脚线：PVC卷材上翻100mm并做圆弧角（医疗级要求）
8.适用：医院/学校/实验室/办公
```

### 4.5 不锈钢踢脚线
```
1.材质：SUS304不锈钢
2.厚度：0.8-1.0mm
3.高度：60mm/80mm/100mm[按设计]
4.基层：9mm阻燃板基层（涂防火涂料）
5.安装方式：结构胶粘贴+玻璃胶收口
6.表面处理：拉丝/镜面/镀钛（黑钛/香槟金）
7.含内阳角加工
```

### 4.6 木地板
```
起地台高度200mm
地板分级：A类实木复合地板
环保等级：E1
表皮厚度：1.5mm
单片尺寸:1232*196mm
接口类型：单锁口
```

### 4.7 复合踢脚线
```
1.踢脚线高度:50mm
2.面层材料种类、规格、颜色:复合材料踢脚线
```

### 4.8 自流平找平+铺塑胶地板
```
1.基层处理：凿除空鼓、松动区域，清理浮灰、油污
2.界面处理剂一道（增加粘结力）
3.水泥基自流平砂浆找平层，厚度50mm
4.自流平施工：搅拌→倾倒摊铺→消泡滚筒排气
5.养护：24h可上人，7天达设计强度，含水率＜8%后方可铺塑胶
6.塑胶地板：同质透心/多层复合PVC塑胶地板
7.厚度：2.0-3.0mm
8.胶水：水性环保专用胶，齿形刮板满刮
9.铺贴：弹线定位，逐块铺贴，滚压排气
10.拼缝：热熔焊接（同色焊线），打磨平整
11.踢脚线：塑胶卷材上翻100mm做圆弧角，或配同色PVC踢脚线
```

## 附录：模板套路

### 写项目特征的通用公式

**装饰类** = 基层材料 + 中间层 + 面层 + 收口方式 + 防火要求

**设备类** = 名称 + 规格型号 + 技术参数 + 安装方式 + 含配件范围

**拆除类** = 拆除对象 + 拆除范围 + 垃圾处理方式

**定制类** = 材质 + 规格尺寸 + 工艺 + "含制作安装" + "满足甲方要求" + "包含全部工作内容"

### 兜底话术
```
1.含制作、安装
2.颜色、样式等满足甲方要求
3.包含完成该项的全部工作内容
```

### 防火通用句
```
所有基层木材均应满足防火要求,涂上规范要求涂刷的遍数、本地消防大队同意使用的防火涂料
```
