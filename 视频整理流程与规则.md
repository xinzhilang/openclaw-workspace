# 视频整理工作流程 v4

> 📅 最后更新: 2026-05-27
> 📍 适用范围: 全电脑所有视频文件

## 核心工作流程

```
本地 OCR (PaddleOCR) → 提取指定帧 → 识别文字
    ↓
能识别 → 过滤水印/广告文字 → 校验标题 → AND/OR 分类 → 重命名 → 移动
    ↓
不能识别 → 留在「其他·未分类」，不动
                ↓
          只有你决定时，才上百度 OCR
```

## 操作步骤

### 1. 启动本地 OCR 服务
```powershell
C:\paddle_ocr\venv\Scripts\python.exe C:\paddle_ocr\ocr_service.py
```
服务运行在 `http://localhost:8866`

- 不能识别的 → 留在 `其他·未分类`，通知你决定是否上百度 OCR

## OCR 帧提取策略（递进帧）

严格按以下顺序取帧，**首个有文字的帧即止**，不要一次性取多帧混合：

```
0s → 0.1s → 0.2s → 0.5s → 1s → 1.5s → 2s
```

- 从 0s 开始，0.1s 步进，越往后步进放大
- **一旦某一帧识别到文字，立即停止**，不再往后取帧
- 避免取到片尾、中间黑屏等无意义帧

## 水印与广告过滤规则

### 水印黑名单（必须过滤，不作为文件名内容）

以下水印/角标/创作者标记识别到时，**跳过不纳入文件名**：

| 类型 | 关键词 |
|------|--------|
| 创作者水印 | 辣逼小新、secaishen、jianjiClub |
| 平台水印 | shufuayi、无他相机、抖音、快手 |
| 其他 | （按需补充） |

### 广告文字跳过规则

以下广告文字识别到时，**整段跳过**，不影响实际内容识别：

- 「更多原创视频进 ***」
- 「t.me/***」
- 各种推广链接、群组邀请

### 过滤方式
- 水印文字和广告文字在合并成文件名前**单独过滤**
- 过滤后的纯内容文字才用于重命名和分类

## 标题校验规则

### 简短校验
- 识别出的文字长度 < 4 个有效字 → 视为无效 OCR，不处理
- 纯数字/日期/符号 → 跳过（如 `2022.1.3` 单独出现）

### 语句通顺检查
- OCR 识别乱序或明显错乱的文本 → 尝试更优帧或延长递进
- 文字断句不合理时 → 回退到未分类，不强行改名

### 重名冲突处理
```
同名文件 → 自动追加编号：
  标题.mp4          （第一个）
  标题_A.mp4        （第二个）
  标题_B.mp4        （第三个）
  ——或——
  标题(1).mp4       （第一个）
  标题(2).mp4       （第二个）
```
编号方式由你选择，默认用 `_A` `_B` 或 `(1)` `(2)`。

### ⚠️ 规则优先级说明

1. AND 规则和 OR 规则**按列表顺序匹配**，前面的优先
2. AND 规则更精确（如"反差婊妈妈"精确匹配到 妈妈·反差）
3. OR 兜底规则放在后面（如"妈妈的教诲"匹配 妈妈·综合）
4. `老婆·出轨` 也支持独立关键词匹配（给老公、背着老公、别人的老婆）
