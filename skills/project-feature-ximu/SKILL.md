## 编写原则

| 要素 | 说明 |
|------|------|
| 材质 | 什么材料：品牌/型号/规格/等级 |
| 做法 | 施工工序，从基层到面层逐层 |
| 位置 | 室内/室外/墙面/地面/天棚 |
| 要求 | 防火/环保/验收标准 |

通用公式：
- **装饰类** = 基层材料 + 中间层 + 面层 + 收口方式 + 防火要求
- **设备类** = 名称 + 规格型号 + 技术参数 + 安装方式 + 含配件范围
- **拆除类** = 拆除对象 + 拆除范围 + 垃圾处理方式
- **定制类** = 材质 + 规格尺寸 + 工艺 + "含制作安装" + "满足甲方要求" + "包含全部工作内容"

兜底话术：
```
1.含制作、安装
2.颜色、样式等满足甲方要求
3.包含完成该项的全部工作内容
```

防火通用句：
```
所有基层木材均应满足防火要求，涂上规范要求涂刷的遍数、本地消防大队同意使用的防火涂料
```

## 墙面工程

### 乳胶漆墙面
```
1.粉刷5mm厚石膏
2.界面处理剂一道（如YJ-302等）
3.内墙耐水腻子2~3遍刮平打磨
4.涂乳胶漆3遍
```

### 木饰面墙面
```
1.木饰面品种、颜色：[按设计]
2.木皮厚度：≥60丝（造型线条≥35丝）
3.基层：12mm厚多层板/细木工板（三防处理：防火+防潮+防腐）
4.安装方式：干挂（卡扣/挂件）
5.三防处理：接触墙体一侧刷防腐涂料，背离墙体一侧及两侧边刷防火涂料
```

### 硬包墙面
```
1.基层：15mm阻燃板（三防处理）
2.填充层：薄海绵或珍珠棉（5-10mm）或无填充
3.面层：皮革/布艺/科技木皮
4.安装方式：阻燃板基层+填充+面料包覆
5.阳角：45°拼接或专用阳角收口条
```

### 木饰面硬包组合墙面
```
项目特征（组合式-分开列项）：
木饰面部分：
1. 龙骨：木龙骨 30×40mm，间距≤400mm（刷防火涂料三遍）
2. 基层：15mm厚阻燃板（三防处理）
3. 面层：木饰面面板WD-XX，木皮厚度≥60丝
4. 收边：不锈钢收边条 60×25mm

硬包部分：
1. 龙骨：木龙骨 30×40mm，间距≤400mm（刷防火涂料三遍）
2. 基层：15mm厚阻燃板（三防处理）
3. 面层：皮革/布艺硬包
4. 收边：与木饰面交接处用不锈钢收口条
```

### 外墙真石漆
```
1.外墙腻子找平2遍
2.抗碱封闭底漆一道
3.真石漆喷涂2遍
4.罩面清漆一道
5.颜色与分格缝：[按设计]
```

## 天棚吊顶

```
1.38系列轻钢龙骨，间隔90-120cm
2.U型轻钢覆面横撑龙骨50*20mm，间距400mm
3.U型轻钢覆面次龙骨50*20mm，间距400mm
4.U型轻钢承载龙骨50*20mm，中距≤1200
5.∮8钢筋吊杆，双向中距≤1200
6.单层9.5mm厚纸面石膏板
7.粉刷5mm厚石膏+界面剂一道
8.耐水腻子2~3遍刮平打磨

```

## 常见问答

### 项目特征写多细？
- **招标阶段**：关键参数 + "[按设计]"
- **结算阶段**：按实际做法描述
- **变更签证**：写变更前后对比

### 防火涂料怎么写？
```
防护材料种类：木材面刷防火涂料三遍
统一加防火通用句
```

### 两材交接怎么写？
```
收边材料：不锈钢收边条 / 铝合金收口线 / 同色结构胶嵌缝
```
